AEPO
================================

`AEPO (Annotation Efficient Preference Optimization) <https://github.com/CyberAgentAILab/annotation-efficient-po>`_ is a Python package for preference dataset preparation.
It implements a set of functions to generate preference datasets from raw data.

.. image:: aepo.png
  :width: 500
  :alt: AEPO logo

Installation
------------
You can install AEPO from PyPI using pip.
Python 3.9 or higher is required.

.. code-block:: bash

    pip install aepo
